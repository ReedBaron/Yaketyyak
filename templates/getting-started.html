Why does this happen?

Apple requires a $99/year developer license to sign apps. Yakety Yak is free and open source, so we haven't paid for that. The app is safe — you can review the entire source code yourself.

Allow it to open (pick one method)

You have two easy options. Pick whichever feels more comfortable:

Option A — Right-click method (easiest)
  1. Right-click (or Control+click) on Yakety Yak.app in Finder
  2. Select "Open" from the context menu
  3. A new dialog appears — this time with an "Open" button
  4. Click "Open" — done!

Open

After right-clicking and choosing "Open", you get this dialog. Click "Open".

Option B — System Settings method
  1. Click the Apple menu in the top-left corner
  2. Open System Settings (or "System Preferences" on older macOS)
  3. Go to Privacy & Security
  4. Scroll down to the Security section
  5. You'll see: "Yakety Yak" was blocked from use because it is not from an identified developer
  6. Click "Open Anyway"
  7. Enter your Mac password when prompted

One-time only: You only need to do this once. After that, macOS remembers your choice and Yakety Yak opens normally every time.

Open your terminal

Press Ctrl+Alt+T or search for "Terminal" in your applications menu.

Extract the archive

Navigate to your Downloads folder and extract the .tar.gz file:

Terminal
~$ cd ~/Downloads
~$ tar -xzf YaketyYak-Lite-Linux.tar.gz

This creates a linux-package/ folder with everything you need.

Run the installer

The package includes an install script that sets everything up for you:

Terminal
~$ cd linux-package
~$ chmod +x install.sh
~$ ./install.sh
Installing Yakety Yak...
Done! Run 'yakety-yak' to start.

Don't want to install? You can run it directly without installing:
chmod +x bin/yakety-yak && ./bin/yakety-yak

Launch Yakety Yak

After installation, just type:

Terminal
~$ yakety-yak
